# videotest
**环境 Node v12.6.0**

查看效果请打开 http://104.248.150.21:8888/ 美国服务器请使用 ss

命令行运行 `node server206` 返回 206  

命令行运行 `node server200` 将返回状态码 200

命令行运行 `node server` 运行 index.html

index.html 查看视频
